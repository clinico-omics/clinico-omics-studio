### Project setup

- Clone repo
```bash
git clone https://github.com/clinico-omics/datains-frontend.git
cd datains-frontend
```

- Install dependencies
```
yarn install
```

- Compiles and hot-reloads for development
```
yarn run serve
```

- Compiles and minifies for production
```
yarn run build
```

- Lints and fixes files
```
yarn run lint
```
